# **App Name**: FAQ-AI Simplified

## Core Features:

- Question Submission: Allow users to submit questions via a form.
- AI-Powered Answering: Use a generative AI tool to provide an answer to each submitted question based on existing documentation. Save the questions/answers into Firestore.
- Admin approval flow: Before the answers are shown, an admin needs to approve the question/answer pair. These are saved to Firestore
- FAQ Display: Display approved questions and answers in a clear, searchable format.

## Style Guidelines:

- Primary color: Light, sky-blue (#87CEEB) evoking trustworthiness and calm.
- Background color: Very light gray (#F0F0F0), close in hue to the primary color but very desaturated.
- Accent color: Muted lavender (#B19CD9), lighter than the primary color for visual contrast and highlighting.
- Font: 'Inter' sans-serif font. Using 'Inter' for both headlines and body text.
- Use simple, clear icons to represent different categories or topics in the FAQ.
- Clean, minimalist layout with clear sections for question submission, admin approval, and FAQ display.
- Subtle transitions when displaying answers or navigating through the FAQ.
